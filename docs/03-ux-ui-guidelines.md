# UX/UI Guidelines

## Tipografía
- **Plus Jakarta Sans**: geométrica, limpia y moderna; mantiene look serio-tech y legible.

## Color & tema
- Fondo claro y limpio.
- Un solo acento azul para CTAs y focus.
- Bordes suaves y sombras discretas.

## Espaciado y tamaños
- Base spacing 4/8/12/16/24/32.
- Inputs y botones con **altura 48px** (táctil mínimo).
- Cards con padding generoso y separación visual.

## Componentes
- **Brand**: texto "ObrApp" con tracking sutil y peso semibold.
- **Button**: primario/secondary, estados hover/disabled/loading.
- **Input**: label + helper/error, focus visible.
- **Card**: superficie con borde y sombra suave.
- **Navbar**: minimal, con user pill y logout.

## Responsive
- Mobile-first, card centrada y full width en móviles.
- Desktop mantiene card centrada, layout respirable.

## Experiencia
- Loading state para rehidratación y submit.
- Errores inline visibles.
- Sin pantalla blanca post-login.
